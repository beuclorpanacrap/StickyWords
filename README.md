StickyWords
Autocomplete and typo-correction system
Type a partial or misspelled word anywhere. Predictions appear instantly. Press 1–4 to pick one. The system learns your habits over time.
Parts
Desktop app — system-wide overlay, captures keystrokes, injects corrections
Chrome extension — brings autocomplete into the browser
Web dashboard — live stats, typo charts, per-app breakdown at localhost:5000
