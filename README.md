StickyWords is an autocomplete and typo-correction system. To use it, type a partial or misspelled word anywhere, afterwards, predictions should appear instantly. Press 1–4 to pick one. The system learns your habits over time. The solution's parts include:


A Desktop app — system-wide overlay, captures keystrokes, injects corrections
Chrome extension — brings autocomplete into the browser
Web dashboard — live stats, typo charts, per-app breakdown at localhost:5000

Additional/Helper files for the app are:

vocab.txt -Main word list
user_frequency.json - Your personal word usage (auto-generated)
usage_stats.json - YCorrection history (auto-generated)


Please note that the app is Windows Only!!!
